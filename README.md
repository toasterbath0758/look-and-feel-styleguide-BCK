# look-and-feel-styleguide-BCK READ ME!

## Intro
De beschrijving van de opdracht:
Ontwerp en maak met een team een styleguide om te gebruiken voor een opdracht van een opdrachtgever.

## Beschrijving
De styleguide voor de online nieuws pagina voor de BuurtCampusKrant Amsterdam!

## Kenmerken
De styleguide is gebouwd met HTML en CSS.
De styling is gebaseerd op de fysieke krant van de BuurtCampus.
Interessante dingen die zijn toegevoegd:
De CSS pagina word gelinkt aan de het individuele gedeelte van de opdracht in de persoonlijke REPO's van elk persoon in ons groepje en kan live worden aangepast.
dit is vet omdat je eigenlijk 1 CSS maakt voor 4 verschillende websites! erg cool!
